三层职责划分

1

Format Output 层

Chat/RAG/Search × presentation-html/html-renderer/step-by-step-tutor/framework-extraction

每个组合跑 agent → 截取最终 answer → Playwright 渲染 → 截图保存

2

Ingestion-Answer 层

PDF 上传 → 解析/分块 → embedding → Milvus 入库 → RAG 检索 → answer

验证 ingestion 后的文档能被 retrieval 命中，answer 引用来源正确

3

Regression Report 层

读取前两层的落盘结果 → 生成 Markdown/HTML 对比报告

同一 query 在不同 strategy/format 下的输出并排展示

数据流

Query → AgentRequest → StrategyExecutor
                           │
                           ├── LLM calls (recorded by RecordingLlmProvider)
                           ├── Tool calls (recorded by CollectingSink)
                           │
                           ↓
                      AgentRunResult
                           │
              ┌───────────┴───────────┐
              ↓                       ↓
        text answer             HTML answer
              │                       │
              ↓                       ↓
         answer.txt            playwright_helper::render()
                                   │
                                   ↓
                              screenshot.png
                                   │
                                   ↓
                        result_serializer::save()
                                   │
                                   ↓
              e2e_output/{timestamp}/{test_name}/
                                   │
                                   ↓
              e2e_regression_report.rs 读取所有结果
                                   │
                                   ↓
                              report.md
