Context OS Main Agent：Query → Answer 的 Behavior Skill 装载机制

基于当前代码实现：RAG 与 Chat 两种模式。Behavior Skill 是 prompt envelope 内的静态行为 profile，不是动态工具系统。

Chat / General

  • system: GENERAL_SYSTEM_PROMPT
  • skill: general_chat
  • context: memory + recent turns + preferences
  • output: natural-language answer, no citations

RAG Plan

  • system: RAG_PLAN_SYSTEM_PROMPT
  • skill: rag_plan
  • context: doc_scope + doc metadata
  • output: ExecutePlanRequest JSON or clarify

RAG Answer

  • system: RAG_ANSWER_SYSTEM_PROMPT
  • skill: rag_answer
  • context: retrieval bundle + coverage + backend trace
  • output: streamed answer + citations + done
