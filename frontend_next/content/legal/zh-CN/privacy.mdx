# 隐私政策

最后更新: 2026-06-13 · 版本: 2026-06-13

Context-OS（以下简称"我们"或"本服务"）非常重视您的个人信息保护。本隐私政策详细说明我们如何收集、使用、存储和保护您的个人信息。在使用本服务前，请仔细阅读本政策。

## 1. 我们收集的信息

### 1.1 账号标识信息

- **收集内容：** 电子邮箱地址、显示名称（可选）
- **处理目的：** 账号注册、身份认证、密码找回、服务通知
- **存储方式：** 加密存储于 PostgreSQL 数据库
- **保留期限：** 账号存续期间；注销后按法律要求保留必要期限后删除

### 1.2 上传文档与元数据

- **收集内容：** 您上传的文档文件（PDF、文本等）、文件名、上传时间、文件大小等元数据
- **处理目的：** RAG 文档处理、文本提取、向量索引、检索问答
- **存储方式：** 文件存储于对象存储服务（生产环境使用云 S3/OSS），元数据及解析结果存储于 PostgreSQL
- **保留期限：** 账号存续期间；您可随时删除已上传的文档
- **是否用于模型训练：** **不会。** 您上传的文档不会被用于 AI 模型的训练、微调或改进

### 1.3 聊天与推理内容

- **收集内容：** 您与 AI 的对话记录、提问内容、AI 生成的回答
- **处理目的：** 提供对话服务、上下文记忆（若开启）、服务质量改进
- **存储方式：** 存储于 PostgreSQL 数据库
- **保留期限：** 账号存续期间；您可随时在应用内删除对话记录
- **导出方式：** 您可以通过设置页面导出您的对话历史

### 1.4 向量 Embedding

- **收集内容：** 由文档内容转换生成的数学向量表示
- **处理目的：** 实现语义检索功能，支持相似性搜索
- **存储方式：** 存储于 Milvus 向量数据库
- **技术说明：** 向量 embedding 是文档内容的数值化表示，不包含可直接识别的个人信息。删除原始文档时，对应的向量数据将同步删除。

### 1.5 使用与计费信息

- **收集内容：** API 调用次数、存储用量、功能使用情况、订阅状态
- **处理目的：** 配额管理、账单计算、服务优化
- **存储方式：** 存储于 PostgreSQL 数据库
- **第三方交互：** 支付信息由 Creem 或支付宝处理，我们仅接收支付状态（成功/失败）和必要的交易标识，不存储您的完整支付卡信息

### 1.6 第三方 AI/OCR 服务

为提供核心功能，我们会将部分数据传输至以下第三方服务提供商：

| 服务提供商 | 用途 | 数据类型 |
|-----------|------|----------|
| DeepSeek | AI 推理 | 对话内容、文档上下文 |
| DashScope（阿里云百炼） | 文本嵌入、多模态处理 | 文档文本、查询内容 |
| Paddle OCR | 文档文字识别 | 文档图像 |
| Brave Search | 网络检索增强 | 搜索查询（仅开启联网搜索时） |

- **委托处理：** 上述数据传输仅限于向您提供本服务所必需的范围
- **跨境传输：** 部分服务提供商的服务器可能位于中国大陆境外。我们会确保数据传输符合适用的数据保护法律要求
- **安全保障：** 我们通过 API 调用方式使用上述服务，数据传输采用加密协议

### 1.7 日志与审计信息

- **收集内容：** 访问日志（IP 地址、访问时间、请求路径）、错误日志、安全审计事件
- **处理目的：** 服务运维、故障排查、安全防护、合规审计
- **存储方式：** 服务器日志文件
- **保留期限：** 30 天自动轮转清理；安全审计日志保留至必要期限

### 1.8 设备与浏览器信息

- **收集内容：** 浏览器类型、操作系统、屏幕分辨率、语言偏好
- **处理目的：** 优化界面适配、排查兼容性问题
- **存储方式：** 客户端 cookie 或 localStorage（非追踪用途）

## 2. 信息使用原则

我们遵循以下原则处理您的个人信息：

- **合法正当：** 仅在向您提供服务所必需的范围内收集和使用
- **最小必要：** 不收集与服务无关的个人信息
- **目的限制：** 不将您的信息用于本政策未载明的其他目的
- **安全保障：** 采取合理的技术和管理措施保护信息安全

## 3. 信息共享与披露

### 3.1 不出售

我们不出售您的个人信息。

### 3.2 共享情形

我们仅在以下情形中与第三方共享您的信息：

- **服务提供商：** 为提供服务所必需的技术合作伙伴（详见 §1.6）
- **法律要求：** 法律法规、诉讼或政府主管部门的要求
- **安全保护：** 为保护我们、您或公众的权利、财产或安全所必需
- **业务变更：** 在合并、收购或资产出售的情况下，信息可能作为业务资产转移（我们会提前通知您）

### 3.3 匿名化数据

我们可能将您的信息进行匿名化或聚合处理后用于统计分析、产品改进等目的。匿名化处理后的数据不再能够识别特定个人。

## 4. 数据安全

### 4.1 技术措施

- 传输层加密（HTTPS/TLS）
- 数据库访问控制与加密
- API 认证与授权机制
- 定期安全审计与漏洞扫描

### 4.2 管理措施

- 最小权限原则：员工仅能访问其工作所必需的数据
- 安全意识培训
- 安全事件应急响应流程

### 4.3 安全事件

如发生个人信息安全事件，我们将按照法律法规要求及时通知受影响的用户，并采取补救措施。

## 5. 您的权利

根据适用的个人信息保护法律（包括但不限于《个人信息保护法》《数据安全法》），您享有以下权利：

### 5.1 查阅权

您有权查阅我们收集的您的个人信息。

### 5.2 复制权

您有权获取您的个人信息副本。

### 5.3 更正权

您有权要求更正不准确的个人信息。

### 5.4 删除权

在以下情况下，您有权要求删除个人信息：

- 处理目的已实现或信息不再必要
- 您撤回同意
- 我们违反法律规定处理信息

### 5.5 注销账号

您可以通过以下方式注销账号：

- **自助注销：** 在设置页面提交注销申请
- **联系客服：** 发送邮件至 [legal@context-os.com](mailto:legal@context-os.com)

注销后，我们将在合理期限内删除或匿名化您的个人信息，法律法规另有规定的除外。

### 5.6 撤回同意

对于基于同意的个人信息处理，您有权随时撤回同意。撤回同意不影响撤回前已进行的处理的合法性。

### 5.7 行使方式

行使上述权利的途径：

- **应用内操作：** 部分权利可在设置页面直接操作
- **邮件联系：** 发送请求至 [legal@context-os.com](mailto:legal@context-os.com)
- **响应时间：** 我们将在 15 个工作日内响应您的请求

## 6. Cookie 与本地存储

本服务使用必要的 cookie 和 localStorage 以实现以下功能：

- **会话管理：** 维持您的登录状态
- **偏好设置：** 记忆您的界面语言、主题等偏好
- **安全防护：** CSRF 防护等安全机制

我们不使用第三方追踪 cookie。您可以通过浏览器设置管理或清除本地存储数据。

## 7. 未成年人保护

本服务不面向 14 周岁以下的未成年人。如您为未成年人，请在监护人指导下使用本服务。如果我们发现已收集了未成年人的个人信息，将在核实后尽快删除。

## 8. 隐私政策更新

### 8.1 更新方式

- **一般修改：** 更新本页面并修改"最后更新"日期
- **重大变更：** 通过电子邮件或产品内通知等方式提前告知

### 8.2 您的选择

隐私政策更新后，您继续使用本服务即表示您同意更新后的政策。如您不同意更新后的政策，应停止使用本服务并注销账号。

如您对本隐私政策有任何疑问、意见或请求，请通过以下方式联系我们：

- **邮箱：** [legal@context-os.com](mailto:legal@context-os.com)
